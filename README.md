# Casa Norte 🤠

**Cinematic teaser landing page for Casa Norte — suede leather cowboy boots, handcrafted in Mexico.**

## Features
- Full-screen cinematic slideshow with slow zoom effect
- Film grain texture overlay
- Bold serif typography (Playfair Display)
- Email sign-up / waitlist form
- Scroll-triggered fade-in animations
- Fully responsive (mobile + desktop)

## Tech Stack
- Pure HTML / CSS / JavaScript (zero dependencies)
- Google Fonts: Playfair Display + Inter
- Unsplash placeholder images (replace with real product shots)

## Getting Started

Just open `index.html` in a browser — no build step needed.

To replace images, update the `background-image` URLs inside the `.slide` divs in `index.html`.

To connect the email form to a real service (Mailchimp, ConvertKit, etc.), update the fetch call in `script.js`.

## Deploy
Deploy instantly with [GitHub Pages](https://pages.github.com/), [Netlify](https://netlify.com/), or [Vercel](https://vercel.com/).
